# Seasonal Shopify Banner

A reusable Shopify section for a responsive promo banner with local weather API.

## Features
- Headline, subtext, button
- Optional image
- Color controls
- Mobile-friendly layout

## Installation
1. In Shopify Admin: Online Store → Themes → Edit code
2. Add a new file: `sections/promo-banner.liquid`
3. Copy/paste the contents from this repo:
   - `sections/promo-banner.liquid`
4. Add `assets/promo-banner.css` and `assets/promo-banner.js`
5. Theme Editor → Add section → “Promo Banner”
